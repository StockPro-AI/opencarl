  <name>task 2: Create agents-writer.ts with integration logic</name>
  <files>src/integration/agents-writer.ts</files>
  <action>
Create src/integration/agents-writer.ts implementing reversible AGENTS.md integration using HTML comment markers.

Export two async functions:

1. `integrateCarl(agentsPath: string, carlDocsPath: string): Promise<{success: boolean; message: string}>`
   - Read CARL-AGENTS.md content from carlDocsPath
   - Wrap with markers: `<!-- CARL-START - DO NOT EDIT -->` and `<!-- CARL-END - DO NOT EDIT -->`
   - If AGENTS.md exists and already contains CARL-START marker, return success with "already integrated" message
   - Otherwise, append CARL section to end of AGENTS.md (create file if needed)
   - Always use UTF-8 encoding, no BOM
   - Return success/failure with descriptive message

2. `removeCarlIntegration(agentsPath: string): Promise<{success: boolean; message: string}>`
   - Read AGENTS.md content
   - Find CARL-START and CARL-END markers
   - If not found, return success with "no integration found" message
   - Remove entire CARL section including markers
   - Clean up whitespace (preserve single blank line between remaining content)
   - Write updated content back
   - Return success/failure with descriptive message

Use fs/promises for all file operations. Handle file not found gracefully in removeCarlIntegration.
</action>
